# UI revamp demo — Masters' Union brand

Private preview at `/labs/ui-demo`. Tokens copied from the Masters' Union marketing design system
(see `mu-design-tokens.md` + `Downloads/Masters Union Marketing Site.html`).

## Exact brand colors

| Token | Hex | Use |
|-------|-----|-----|
| Sky | `#39B6D8` | Eyebrow bar, focus, sky washes |
| Orange | `#E38330` | Secondary accent |
| Yellow | `#F7D344` | Signature CTAs, scribble, selection |
| Black | `#090909` | Primary text / primary buttons |
| White | `#FFFFFF` | Page canvas |

Washes: `#EAF7FB` / `#FBEEE3` / `#FEF8E0`

## Type

- **Galano Grotesque** brand primary (not bundled)
- **Montserrat** official public fallback via `next/font`
- Display: weight **800**, tight tracking (−0.03em)

## Shape

- Cards: **12px** radius, 1px `#E5E5E5` border, light shadow
- Buttons: **8px** radius, 1.5px border (MU, not Sana pills)
- Max width **1200px**

## Patterns

- Eyebrow: uppercase + **18×2px sky bar**
- Scribble highlight behind key hero word (yellow bar)
- Accent button = yellow fill + dark text
- Primary button = black fill + white text
