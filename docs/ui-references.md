# UI references (Release 1)

Mood / inspiration for clarity pass — not a full redesign.

## References

1. [Linear](https://linear.app) — sparse nav, strong hierarchy
2. [Vercel Dashboard](https://vercel.com/dashboard) — calm density, clear status chips
3. [Notion](https://www.notion.so) — readable lists, soft borders
4. [Supabase](https://supabase.com/dashboard) — table + summary split
5. [Stripe Dashboard](https://dashboard.stripe.com) — summary cards above detail
6. [Raycast](https://www.raycast.com) — keyboard-first search patterns
7. [Arc](https://arc.net) — sectioned sidebars

## Adopted R1 tokens

1. **Summary-first**: counts / status cards before long tables (Folders).
2. **Single amber accent**: keep existing DFI amber; avoid multi-accent chrome.
3. **Airier cards**: prefer `space-y-6` section rhythm and hide bulk queues behind modals.

Apply these on Folders, Find (Search / Video Carousel), and Indexing Errors — not a ground-up visual rewrite.
